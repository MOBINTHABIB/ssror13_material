four weeks at a glance

  1. Intro to Ruby on Rails
  2. More Ruby on Rails
  3. Work on Project (part 1)
  4. Work on Project (part 2)

intro to RoR

learn the basics of developing web apps with Ruby on Rails

  • Introduction to web applications
  • Intro to Ruby
  • Intro to Rails
  • Your first microapp

more RoR

get to know the components of a RoR web app

  • Views Layout customization
  • Models and DB migrations
  • Controllers and routes
  • Useful gems

work on project (part 1)

start working on a real project from the ground up

  • Agile software development and SCRUM
  • Presentation of the project
  • Requirements
  • Design of basic features and data model
  • Implementation of basic features and data model

work on project (part 2)

more advanced stuff and APIs for third party mobile apps

  • Advanced features design and implementation
  • JSON API

follow-up opportunities

  • collaborate with MLL on projects for Mozambique
  • collaborations with

  • Scholarships for courses and internships at University of Trento (duration: 6 months - more details soon)
